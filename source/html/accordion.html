Обязательные data атрибуты

data-accordion="parent" Устанавливается на родителе

data-accordion="element" Устанавливается на элементе аккордеона

data-accordion="button" Устанавливается на кнопке открытия

data-accordion="content" Устанавливается на обёртке контента

Не обязательные data атрибуты

data-type="single" Устанавливается на родителе, отвечает за показ только одного открытого аккордеона в группе родителя

is-open Это класс которые можно повесить на элемент аккордеона для его открытия при загрузке.

Аккордеоны можно вклыдывать друг вдруга с любой глубиной вложенности. Но в данный момент анимированное открытие возможно только в аккордеонах без вложенности, но мы над этим работаем)
