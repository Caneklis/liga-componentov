Подсветка пунктов навигации при скроле

Скачать компонент

Лок фокуса состоит из из следующих файлов: scroll-translator.js и scroll-translator.scss js обязателен для переноса в проект, из файла scroll-translator.js экспортится класс ScrollTranslator, в main создаётся экземпляр класса и у него вызывается метод init()

              
                import {ScrollTranslator} from '../components/scroll-translator/js/scroll-translator';
                const scrollTranslator = new ScrollTranslator();
                scrollTranslator.init();

Обязательные дата атрибуты

data-translator="parent" Добавляется к родителю

data-translator-type="" Добавляется к родителю, отвечает за то как будет считаться изменение индикатора при скроле горизонтально или вертикально, может иметь два значения horizontal или vertical

data-translator="indicator" Добавляется к индикатору
