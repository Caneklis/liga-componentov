Подсветка пунктов навигации при скроле

Скачать компонент

Подсветка пунктов навигации состоит из из следующих файлов: navigation-changer.js js обязателен для переноса в проект, из файла navigation-changer.js экспортится класс NavigationChanger, в main создаётся экземпляр класса и у него вызывается метод init()

              
                import {NavigationChanger} from '../components/navigation-changer/js/navigation-changer';
                const navigationChanger = new NavigationChanger();
                navigationChanger.init();

Обязательные дата атрибуты

data-navigation-link Добавляется к ссылке

Так же обязательно связывать ссылки с блоками с помощью href в ссылке и id в секции

              
                <a href="#section-1" class="navigation__link" data-navigation-link>Секция 1</a>
                <section class="section-navigation" id="section-2"></section>

По умалчанию подключен MoveTo
