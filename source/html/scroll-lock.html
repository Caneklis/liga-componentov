Лок фокуса

Скачать компонент

Лок фокуса состоит из одного файла scroll-lock.js он обязателен для переноса в проект, из файла экспортится класс FocusLock, в файле создаётся экземпляр класса и записывается в глобальную область видимости window.scrollLock = new ScrollLock();

Пример подключения 

                
                  import '../components/scroll-lock/js/scroll-lock';

В проекте пользуйтесь глобальной переменной window.scrollLock

В классе есть два публичных метода

disableScrolling() Блокирует скролл

enableScrolling() Разблокирует скролл
