Файлы необходимые при поддержке на проекте ie

Скачать компонент

Описание

В архиве лежат следующие файлы:

ie-fix.js файл с различными полифилами, при необходимости закинуть файл в парку utils вашего проекта. Пример подключения: https://skr.sh/sCAPulmJkcT?a

polyfill-object-fit.js файл с полифилом object-fit, при необходимости закинуть файл в парку utils вашего проекта. Пример подключения: https://skr.sh/sCAJMSpibqr?a

swiper.js файл с Swiper версии 5.3.6

swiper.scss scss файл для Swiper версии 5.3.6

При необходимости нужно заменить оба файла и scss и js. Пример подключения: в файле swiper.scss меняем всё содержимое https://skr.sh/sCAASBKH5hs в файле swiper.js меняем всё содержимое https://skr.sh/sCA8E4UxeZQ?a и не забываем указывать версию пакета https://skr.sh/sCAKDupGwj1?a
