Табы

Скачать компонент

Табы состоят из следующих файлов: tabs.js, init-tabs.js, data-tabs.scss все файлы обязательны к переносу в проект! Файл tabs.scss добавлен только для стилизации, в нём вы можете менять всё что угодно

В данной реализации используется ResizeObserver что помогает менять высоту блока при изменении размеров его содержимого без вызова дополнительных методов или функций (например если в табе будет кнопка показать ещё которая будет разворачивать дополнительный текст)

В файле init-tabs.js создаётся экземпляр класса Tabs который записывается в переменную tabs, им вы можете пользоваться по проекту при необходимости, так же он записывается в window.tabs но запись закомментирована раскомментировать только при запросе бэкэнда на вынос экземпляра класса в глобальную область видимости!

Обязательные data атрибуты

data-tabs="parent" Устанавливается на родителе

data-tabs="controls" Устанавливается на родителе контролов (кнопок)

data-tabs="control" Устанавливается на контроле (кнопоке)

data-tabs="content" Устанавливается на родителе показываемых элементов

data-tabs="element" Устанавливается на показываемом элементе

Не обязательные data атрибуты

data-delay="время в милисекундах" Устанавливается на родителе отвечает за задержку при смене табов, а имено задерку между убиранием активного класса у активного элемента и установкой активного класса на выбранном элементе

data-height="max" Устанавливается на родителе, если он установлен, то высота блока будет равна максимальной высоте элементов таба

Методы экземпляра классса tabs

tabs.openTab(control) принимает контрол (кнопку) в виде дом элемента отвечающую за открытие таба

tabs.reInit() Вызывается при динамическом добавлении новых табов, никак не влияет на те табы что уже есть на странице

Инициализация

По умолчанию открыт первый элемент таба, все классы расставляются автоматом. Если нужно что бы был открыт не первый элемент таба, то нужно установить класс is-active на нужный вам контрол (кнопку). Если вы установите этот класс на всех контролах (кнопках), то он снимется со всех контролов (кнопок) кроме первого по порядку в HTML относящихся к своему родителю (data-tabs="parent"), во вложенных родителях (data-tabs="parent") произойдёт тоже самое но относительно своего родителя
